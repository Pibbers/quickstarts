Anypoint Connector for Teradata (Teradata コネクタ) は、Mule アプリと Teradata Vantage データベース間の通信を確立し、Teradata Vantage インスタンスに接続してデータをロードし、Teradata Vantage テーブルで SQL クエリーを実行できるようにします。

リファレンス: xref:reference.adoc[Teradata コネクタ リファレンス]

リリース ノート: xref:release-notes.adoc[Teradata コネクタ リリース ノート]

== 始める前に

このコネクタを使用するには、以下のことを理解しておく必要があります。

* Teradata Vantage SQL
* Anypointコネクタ
* Mule ランタイム エンジン (Mule)
* Mule フローの要素とグローバル要素
* Anypoint Studio (Studio)

アプリを作成する前に、次のものが必要です。

* Anypoint Studioバージョン7.5以降
* Teradata Vantage ターゲット リソースにアクセスするための信頼証明

include::ROOT:partial$vantage_clearscape_analytics.adoc[]

== コネクタの一般的なユースケース

Teradata コネクタを使用すると、以下のことが可能になります。

* 事前定義されたクエリー、動的に構築されたクエリー、および自己完結型でカスタマイズ可能なテンプレート クエリーを実行します。
* ソース リスナー操作を使用して、フローのデータ ソース セクション内のデータベースから読み込みます。
* プロセス セクションの任意の場所で、データベースの読み取りと書き込みを行う他の操作を実行します。
* 単一の一括更新を実行して、複数の SQL文を実行します。
* データ定義言語 (DDL) リクエストを作成します。
* ストアド プロシージャとSQLスクリプトを実行します。

Teradata コネクタは以下をサポートします。

* 接続プーリング
* タイムアウト後の自動再接続
